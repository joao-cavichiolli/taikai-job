# Taikai Community Jobs (Simple)

Public jobs dashboard aggregating:
- Remotive API
- ArbeitNow API

Includes:
- Search / filters
- Web3-only checkbox (keyword filter)
- Light/Dark mode toggle (single button)

## Run
```bash
npm install
npm run dev
```
Open http://localhost:3000/jobs


## Featured Jobs
Featured jobs are detected automatically (hot keywords + last 7 days).


## Filters
- Category (Front-end, Backend, Full Stack, DevOps, IT Support, Non-Technical)
- Has salary (best effort; depends on source)
